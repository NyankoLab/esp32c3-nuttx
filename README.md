# NuttX for ESP32-C3

## Feature
- [x] Binary Loader for ELF
- [x] Hardware AES
- [x] Hardware SHA
- [x] HTTP
- [ ] HTTPS
- [x] MCUBoot Agent
- [x] MCUBoot Loader
- [x] MQTT
- [x] uip WebClient
- [x] uip WebServer

|Type   |Address |File|
|-------|--------|----|
